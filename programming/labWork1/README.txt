2) export PATH=$PATH:.
   для запуска без ./

libleaN3248_1.so содержит опцию float-bin. Используйте ее, если необходимо найти число в его символьном представлении, то есть
1.2 (10) = 01011001100110011001100111111100 (2)
Значит текст dfghjjhgfdcvb01011001100110011001100111111100kjhggbn01 подходит под условия

libleaN3248_2.so содержит опцию float-in-bin. Используйте ее, если необходимо найти число в его битовом представлении в битовом файле, то есть
число представляется в виде четырех октетов, каждый переводится в символ, таким образом получается строка из четырех символов. 
Если написать в файле.с
float result = 1.3;
скомпилировать этот файл, то исполняемый файл будет подходить под условие
