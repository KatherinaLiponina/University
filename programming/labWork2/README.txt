Собирать сервер
make

Собрать клиента
gcc -o client client.c

Запуск стандартный

linkedlist.c, linkedlist.h нужны для main.c, который и есть сам сервер
Между запусками данные сохраняются в /tmp/numbers.txt
Команды есть трех видов
GET N
SET <num> <num> <num>...
DELETE <num> <num> <num>...
В клиент записываются в таком виде после запуска (ключи только для порта и хоста и они обязательны)
На сервере обрабатывается как
GET\n
N\n и т.п
На <num> есть проверка на double, у N - int > 0
Нельзя удалять несуществующие, добавлять существующие
Порт по умолчанию 1255

Единственное, написала демонизацию очень странно. На лекции говорили про new-style демонов, но в мануале все совсееем непонятно, так что here we are

Пример
./server -l file.txt -a 127.0.0.3 -p 1256 -d

./client -a 127.0.0.3 -p 1256
Введите команду: SET 3.14
OK

./client -a 127.0.0.3 -p 1256
Введите команду: SET 1.4142 2.71 32
OK

./client -a 127.0.0.3 -p 1256
Введите команду: GET 3
2.71 3.14 3.14

./client -a 127.0.0.3 -p 1256
Введите команду: DELETE 3.14
OK
